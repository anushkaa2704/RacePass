# RacePass Frontend

This is the user interface built with React + Vite.

## What's in here?

```
frontend/
├── src/
│   ├── components/     # Reusable UI pieces (buttons, forms)
│   ├── pages/          # Full screens (Signup, Dashboard)
│   ├── utils/          # Helper functions (Web3, API calls)
│   ├── App.jsx         # Main app with routing
│   └── main.jsx        # Entry point
├── index.html          # HTML template
├── package.json        # Dependencies
└── vite.config.js      # Vite configuration
```

## How to Run

```bash
cd frontend
npm install
npm run dev
```

Then open http://localhost:5173
